# 🎮 Simon Say Game

A fun memory game built using **HTML, CSS, and JavaScript**.
Repeat the color pattern shown by the game to win!

---

## 🚀 Live Demo

👉 https://github.com/HarshVashishtha19/Wiggles.git

---

## 📌 Features

* Interactive memory game
* Random color sequence
* Increasing difficulty level
* Simple and clean UI

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## ▶️ How to Run Project

1. Clone the repository
2. Open folder in VS Code
3. Open `index.html` in browser or Live Server

---

## 📂 Project Structure

simon-say-game/
├── index.html
├── style.css
├── script.js
└── README.md

---

## 🎮 How to Play

1. Press any key to start
2. Watch the color pattern
3. Repeat the pattern
4. Each level gets harder
